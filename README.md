A Rust implementation of [RSMQ](https://smrchy.github.io/rsmq/about/) (Redis Simple Message Queue).

## Installation

Add this line to your application's Cargo.toml:

```toml
[dependencies]
rsmq = "*"
```

## Usage

```rust
extern crate rsmq;

use rsmq::*;

fn main() {
  println!("TODO");
}
```
